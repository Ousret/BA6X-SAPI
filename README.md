# BA6X-SAPI
Communicate with Wincor Nixdorf displays like BA63 USB (Socket API)

> **How to:**

> - make
> - ./display
> - For PoC, use Telnet at 127.0.0.1 with port 3363

#### <i class="icon-file"></i> Prerequisites

- libhidapi
- libusb
- pthread
